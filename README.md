# Hospital-Appointment-Management-System

The Hospital Appointment Management System is a web-based application designed to streamline the process of managing patients, doctors, and appointments. The system provides an efficient platform for scheduling appointments, maintaining medical records, and reducing administrative workload through database-driven automation,using Python, PostgreSQL, HTML,CSS,and SQL.

#Features
Patient Management
Register new patients,doctors and admin
View patient information
Maintain patient records
Doctor Management
Store doctor details and specialization information
Appointment Scheduling
Book appointments between patients and doctors
View upcoming appointments
Database Integration
Secure storage of patient, doctor, and appointment data
Efficient querying and retrieval using PostgreSQL
Maintenance of relational integrity through foreign key constraints
